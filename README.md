# Greenapple0101

GitLab CI/CD와 Kubernetes를 중심으로 배포 자동화, 운영 안정화, 모니터링을 공부하는 DevOps 엔지니어입니다.

백엔드는 서비스 운영 구조를 더 잘 이해하기 위해 Spring Boot를 학습하고 있습니다.  
관심사는 코드를 직접 많이 작성하는 것보다, 서비스가 안정적으로 빌드되고 배포되고 관찰되는 흐름을 만드는 데 있습니다.

- [rag-project](https://github.com/Greenapple0101/rag-project)
